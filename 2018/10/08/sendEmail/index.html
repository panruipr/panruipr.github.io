使用java工具类email基于smpt（简单邮件传输协议）实现邮件发送

1.设置session属性

//设置属性
Properties prop = new Properties();
prop.setProperty("mail.host","xxxx.com");//使用邮件服务器
prop.setProperty("mail.transport.protocol", "smtp");//协议为SMPT协议
prop.setProperty("mail.smtp.auth", "true");//是否需要验证

2.获取session对象

获取session对象这里有两中方式Session.getInstance(prop)和Session.getInstance(props, authenticator);
1.Session.getInstance(prop)。（本文以这种方式连接，貌似没有授权）
使用这种方式需要获取Transport对象调用connect方法；

Session session = Session.getInstance(prop);
    //开启Session的debug模式，这样就可以查看到程序发送Email的运行状态
    session.setDebug(true);
    //2、通过session得到transport对象
    Transport ts = session.getTransport();
    //3、使用邮箱的用户名和密码连上邮件服务器，发送邮件时，发件人需要提交邮箱的用户名和密码给smtp服务器，用户名和密码都通过验证之后才能够正常发送邮件给收件人。
    ts.connect("xxxx.com", "user","password");
   //ts.connect("xxxx.com","port", "user","password");带服务器端口
2.Session.getInstance(props, authenticator);（授权信息）
在创建邮件会话的时候将授权信息带进去

prop.setProperty(“mail.user”, “xxxxxxx@qq.com“);//用户名
prop.setProperty(“mail.password”, “xxxxxxxxxxxxxxxxxxx”);//授权码
// 构建授权信息，用于进行SMTP进行身份验证
Authenticator authenticator = new Authenticator() {
protected PasswordAuthentication getPasswordAuthentication() {
String userName = prop.getProperty(“mail.user”);
String password = prop.getProperty(“mail.password”);
return new PasswordAuthentication(userName, password);
}
};
// 使用环境属性和授权信息，创建邮件会话
Session mailSession = Session.getInstance(prop, authenticator);
